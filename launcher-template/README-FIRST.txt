Добро пожаловать в Blacked Aeronautics!

Для Portable переместите папку «BA» из архива в C:\Games: итоговый путь должен быть C:\Games\BA.
Не запускайте сборку из дополнительной папки, которую Windows может создать по имени ZIP-файла.

1. Запустите «Blacked Aeronautics.exe».
2. Войдите в аккаунт или укажите свой ник.
3. Выберите Blacked Aeronautics и нажмите «Играть».

Первый запуск займёт больше времени: игра и моды скачаются автоматически.
Дальше Blacked Aeronautics будет сам предлагать новые версии, а перед игрой обновлять моды.
