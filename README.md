# Blacked Aeronautics

Готовый лаунчер Minecraft для Windows. Скачайте его, войдите в аккаунт и нажмите «Играть» — всё остальное загрузится само.

## Скачать

- [Установщик](https://github.com/BlackSoul1337/Blacked-Aeronautics/releases/download/v1.1.2-ely.1/Blacked-Aeronautics-1.1.2-ely.1-win-x64-setup.exe) — самый простой вариант. Лаунчер появится в меню «Пуск».
- [Версия без установки](https://github.com/BlackSoul1337/Blacked-Aeronautics/releases/download/v1.1.2-ely.1/Blacked-Aeronautics-1.1.2-ely.1-win-x64-portable.zip) — распакуйте ZIP и перенесите получившуюся папку в удобное место.

## Как начать играть

1. Откройте Blacked Aeronautics.
2. Войдите через Microsoft или Ely.by. Можно также просто указать свой ник.
3. Выберите Blacked Aeronautics и нажмите «Играть».

Первый запуск может занять некоторое время: лаунчер скачает игру и моды. В дальнейшем он будет сам проверять обновления перед запуском.

Устанавливать дополнительные программы не нужно — всё необходимое уже есть в комплекте.

## Как удалить

Установленную версию можно удалить через список приложений Windows. Версию без установки достаточно удалить вместе с её папкой.
