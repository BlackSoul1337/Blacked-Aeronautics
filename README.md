# Blacked Aeronautics

Готовая Windows x64-сборка Minecraft 1.21.1 / NeoForge 21.1.229 с автоматическим обновлением через packwiz.

Лаунчер основан на PineconeMC (бывший ElyPrismLauncher), но этот проект не связан с PineconeMC, Prism Launcher, Ely.by, Mojang или Microsoft и не одобрен ими.

## Как начать играть

1. Рекомендуемый вариант — скачайте установщик `Blacked-Aeronautics-1.1.2-ely.1-win-x64-setup.exe` из [единого Release](https://github.com/BlackSoul1337/Blacked-Aeronautics/releases/tag/v1.1.2-ely.1) и запустите его. Он установит лаунчер без прав администратора и создаст ярлык в меню «Пуск».
2. В том же Release находится portable-архив `Blacked-Aeronautics-1.1.2-ely.1-win-x64-portable.zip` с готовой папкой лаунчера. Извлеките её и перенесите целиком в удобное место.
3. Запустите лаунчер, добавьте Microsoft/Ely.by-аккаунт либо offline-профиль с вашим ником.
4. Выберите **Blacked Aeronautics** и нажмите **«Играть»**.

Перед каждым запуском лаунчер автоматически сверяет сборку с GitHub Pages. Новые и изменённые файлы скачиваются, удалённые из сборки моды удаляются. Если обновление завершилось с ошибкой, Minecraft не запускается с частично обновлённой сборкой.

В дистрибутив уже включён Eclipse Adoptium Temurin JDK 21. Лаунчер использует относительный путь `java/bin/javaw.exe`, поэтому имя пользователя, системная Java и абсолютный путь установки не имеют значения.

При первом запуске также загружаются Minecraft, NeoForge и файлы сборки.

Setup можно полностью удалить через **Параметры Windows → Приложения → Установленные приложения → Blacked Aeronautics**. Удаляются лаунчер, встроенная Java, скачанные моды, аккаунты и локальные настройки. Portable удаляется удалением его папки.
