# Blacked Aeronautics

Готовый лаунчер Minecraft для Windows. Скачайте его, войдите в аккаунт и нажмите «Играть» — всё остальное загрузится само.

## Скачать

Откройте [последний выпуск](https://github.com/BlackSoul1337/Blacked-Aeronautics/releases/latest) и выберите удобный вариант:

- **Установщик** добавит лаунчер в меню «Пуск». Его название заканчивается на `setup.exe`.
- **Версию без установки** достаточно распаковать и перенести в удобное место. Её название заканчивается на `portable.zip`.

## Как начать играть

1. Откройте Blacked Aeronautics.
2. Войдите через Microsoft или Ely.by. Можно также просто указать свой ник.
3. Выберите Blacked Aeronautics и нажмите «Играть».

Первый запуск может занять некоторое время: лаунчер скачает игру и моды. Новые версии лаунчера, моды и настройки он будет находить сам.

Устанавливать дополнительные программы не нужно — всё необходимое уже есть в комплекте.

## Как удалить

Установленную версию можно удалить через список приложений Windows. Версию без установки достаточно удалить вместе с её папкой.
